# springboot-uni-demo
Spring Boot University Demo

This demo utilizes:

* Spring Boot
* Spring MVC
* Spring Data JPA
* MySQL
* JSP's
* Bootstrap

Set up your MySQL schema using the scripts located at:
src/main/db-scripts

Log in as the admin user to add colleges, majors, teachers, courses, and students.

The admin user/password is admin@sju.edu/admin.
